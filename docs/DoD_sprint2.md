| Categoria | Critério de Conclusão | Verificação |
| :--- | :--- | :--- |
| **1. Funcionalidade** | A funcionalidade principal do item está implementada em VisuAlg. | O código executa sem **erros de sintaxe** e não trava o VisuAlg. |
| **2. Lógica e Teste** | O código funciona corretamente para todos os *Dados de Teste Mínimos* (definidos na DoR). | A saída do VisuAlg corresponde ao cálculo manual para o N-ésimo termo e os 50 seguintes (quando aplicável). |
| **3. Qualidade do Código** | O código está **documentado** e **organizado**. | As variáveis e a lógica principal estão explicadas por **comentários claros** dentro do código VisuAlg. |
| **4. Integridade do Sistema** | O código do item foi integrado com sucesso na *branch* de desenvolvimento. | A *Feature Branch* foi **mergiada** (ou seu Pull Request foi aprovado) na branch **`develop`**. |
| **5. Revisão (Se Aplicável)** | O código foi revisado e aprovado por outro membro do Time de Desenvolvimento. | O revisor confirmou a lógica e a conformidade com o DoD. |
